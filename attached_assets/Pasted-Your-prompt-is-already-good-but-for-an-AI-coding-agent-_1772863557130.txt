# Scroll Animation Requirements

Implement **viewport-based animations** triggered by scrolling.

## Entry Animation

Trigger when the element **enters the viewport**.

Suggested animation style:

```
opacity: 0 → 1
translateY: 40px → 0
scale: 0.98 → 1
duration: ~0.6s
easing: ease-out
```

---

## Exit Animation

Trigger when the element **leaves the viewport**.

Suggested animation style:

```
opacity: 1 → 0
translateY: 0 → -20px
duration: ~0.4s
```

Exit animations should be **subtle and fast**.
